Linux常用命令

发表于 2016-12-02   |   分类于 Linux命令   |   阅读次数

Linux常见的命令

1、ls命令
ls -a 除了显示一般文件名外，还会显示隐藏文件
ls -l 需要查看更详细的文件资料
Ls -F 在列出的文件、目录名称后加一符号，对可执行文件添加一个*号，为目录添加一个/号，为符号链接添加一个@号

2、cd 命令
cd - 回到上一次工作的地方
cd ~ 回到家目录
cd/ 回到根目录
cd ../../../../../../../../ 路径较深时，回到根目录
绝对路径和相对路径
相对路径：参照当前目录进行查找
cd ../init.d/
绝对路径：从根目录开始指定，一级一级递归查找，在任何目录下都能进入指定位置
cd /etc/init.d/
目录补全快捷键：Tab
