Hexo中写博客

我刚开始建设完Hexo博客之后想写一个博客，但是不是道怎么写，查了以后需要用命令行，感觉很麻烦有没有简单的写博客的方法。安装hey插件 写博客很方便。

安装hey插件

在hexo文件夹右键-bash，输入以下命令：

1.npm install nihgwu/hexo-hey –save

配置hexo文件夹的_config.ym 文件：

在任意位置添加：

# Admin
admin:
name: hexo
password: hey
secret: hey hexo
expire: 60*1
# cors: http://localhost:3000

使用hey

在bash下输入：hexo s进入本地的网页

http://localhost:4000/admin/

账号：hexo
密码：hey
进入后如下图：

这是一个简便的管理界面，用它发表文章很简单。
新建文件的时候，点加号新建
左边写文章右边预览（写的时候使用markdown格式写的，自己网上查）
保存的时候点右上角的…符号出现下面的界面：

写好分类，标签点更新就ok了
然后传到github上就行。
